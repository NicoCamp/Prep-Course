*Arrays: Una Array es como una lista que puede incluir distintos tipo de datos, como numeros, strings, booleanos,
funciones e incluso otras arrays, o tambien pueden ser listas vacias.
    -Para declara una Array se deben usar corchetes y escribir dentro los datos deseados: **var Array[1, 2, "stirng"].
    -A los datos que estan dentro de estas lista se puede acceder mediante el uso de indices: Array[0], Array[3].
    -Pueden ser manipuladas, accedidas y recorridas mediante es uso de "Metodos" y "Propiedades".
